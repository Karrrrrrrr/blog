Docker指令
```shell
docker ps 
docker images 
docker rm  #(-f) 强制删除
docker rmi #(-f) 强制删除
docker stats 

docker cp from-file to-file // 必须一个是本地 一个是容器 无法跨容器

docker df

docker export 

docker import 
```
